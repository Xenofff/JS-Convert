Lossy vs lossless compression

JPG and WebP can compress photos with adjustable quality (lossy). PNG is lossless, so size reduction is limited — resizing often helps more.

FAQ

Does compressing reduce dimensions?
Not unless you output PNG (slider acts as resize). For JPG/WebP, dimensions stay the same.
What compresses best?
WebP usually provides the best size/quality ratio.
Is there a size limit?
No server limit — only your browser memory and performance.
